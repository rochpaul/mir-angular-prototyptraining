Mods Institutional Repository (MIR)

fertig

MIR ist eine MyCoRe-Software, die mit nur wenigen Anpassungen - so wie sie ist - als Repositorium eingesetzt werden kann.

MIR verwendet das mächtige und bewährte MODS-Datenmodell. Den Kern der Webanwendung bildet MyCoRe mit einer Vielzahl von Funktionen.
